A small, focused converter for point cloud files. It streams every point from disk through a fixed-size pipeline, so memory use stays flat no matter how big the file is — you can convert clouds far larger than your RAM. When a file is too big to open elsewhere, it can downsample to a target size you pick (in MB, GB, or a point count) while preserving the original distribution.

Features

  • Any-to-any conversionRead and write across every supported format through one common internal point model — hub and spoke, not a tangle of pairwise converters.
  • Larger-than-RAM filesA streaming Reader → Transform → Writer pipeline processes one batch at a time, so memory stays constant regardless of file size.
  • Size-targeted downsamplingAim for a target output size — MB, GB, or a point count — and it thins the cloud randomly while keeping the original spatial distribution. Deterministic, so the same input gives the same result.
  • Beginner-friendly UIPick a file, choose an output format and size, choose where to save. No flags, no command line.
